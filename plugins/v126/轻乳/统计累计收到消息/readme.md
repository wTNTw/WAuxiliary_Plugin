# 好友消息统计助手（改进版）

## 📊 统计命令（私聊发送给自己）

| 命令 | 说明 | 示例 |
|------|------|------|
| `/统计 好友wxid` | 查看指定好友的详细统计 | `/统计 wxid_xxxxx` |
| `/统计 列表` | 列出所有有统计的好友及消息总数 | - |
| `/统计 关键词 添加 词` | 添加统计关键词（不区分大小写） | `/统计 关键词 添加 吃饭` |
| `/统计 关键词 删除 编号` | 删除指定编号的关键词 | `/统计 关键词 删除 1` |
| `/统计 关键词 列表` | 查看当前所有关键词 | - |
| `/统计 定时开启` | 启用每日定时消息 | - |
| `/统计 定时关闭` | 禁用每日定时消息 | - |
| `/统计 定时测试` | 立即测试发送定时消息 | - |
| `/统计 帮助` | 显示本帮助 | - |

---

## ⏰ 内置定时消息功能

- **发送时间**：每天凌晨 0:00-6:00（首次收到消息时触发）
- **发送对象**：配置中的指定好友
- **消息内容**：固定内容（需在代码中修改 `NIGHT_MESSAGE`）
- **防重复**：每天仅发送一次，发送后会有通知给自己
- **可靠性**：6小时时间窗口，避免错过0点
