# 微信步数自动增加插件

### 介绍

本插件提供三种自动增加步数的方式：
1. **时间步数**：每分钟随机增加步数（7:00-22:50），每分钟步数范围可通过命令设置
## 概述
AutoStep 在预设活动时段内自动增加步数，支持按时间与消息两种触发方式。本插件提供三种自动增加步数的方式：
1. **时间步数**：每分钟随机增加步数（7:00-22:50），每分钟步数范围可通过命令设置
2. **消息步数**：每次收发消息时自动增加步数，当前步数越小，增加速度越快
3. **保底步数**：22:50时检查当前步数，若小于保底步数则修改为保底步数

步数增长幅度根据当前步数自动调整：
- 4000步以下：3倍增长
- 4000-8000步：2倍增长
- 8000-16000步：1倍增长
- 16000步以上：0.5倍增长

## 用法

1. 在任意聊天窗口输入命令控制功能
1. 在任意聊天窗口输入命令控制功能
2. 每日0点会自动重置步数
3. 功能状态会自动保存，重启插件后保持上次设置


## 命令（即时生效）
- /时间步数开 — 启用时间自动增加  
- /时间步数关 — 关闭时间自动增加  
- /消息步数开 — 启用消息触发增加  
- /消息步数关 — 关闭消息触发增加  
- /步数状态 — 显示当前步数与关键状态（含时间目标）  
- /步数状态all — 显示完整状态（含分配策略、目标等）  
- /改步数 N — 手动设置当前步数为 N  
- /最大消息步数 N — 设置消息触发的目标上限  
- /分钟步数范围 min-max — 设置时间模式下每分钟步数范围  
- /保底步数 N — 设置每日保底步数（0 表示关闭）  
- /时间步数目标 N — 设置需在截止时间前达到的步数目标（0 表示取消）  
- /分配策略 线性|指数 — 将时间目标的分配策略切换为线性或指数

示例：
- /时间步数目标 20000  
- /分配策略 指数  
- /分钟步数范围 4-12

### 注意事项

- 7:00-22:50时间段内自动增加步数，其他时间不增加
- 7:00-22:50时间段内自动增加步数，其他时间不增加
- 保底步数功能在22:50时执行，若当前步数小于保底步数则自动修改为保底步数
- 消息步数功能开启时，今日目标和进度跟随最大消息步数
- 如果当天实际步数大于插件增加的步数，会以实际步数为准
- 插件设置会自动保存，重启后保持上次设置
- 时间步数增加依赖微信后台，请保留微信后台，否则微信步数更新不及时
- 消息步数功能基于消息触发，若微信在后台时不接收消息，则进入前台后一次性处理
