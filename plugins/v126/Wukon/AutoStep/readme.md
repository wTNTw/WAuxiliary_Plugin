# 微信步数自动增加插件

### 介绍

本插件提供三种自动增加步数的方式：
1. **时间步数**：每分钟随机增加步数（7:00-22:50），每分钟步数范围可通过命令设置
2. **消息步数**：每次收发消息时自动增加步数，当前步数越小，增加速度越快
3. **保底步数**：22:50时检查当前步数，若小于保底步数则修改为保底步数

步数增长幅度根据当前步数自动调整：
- 4000步以下：3倍增长
- 4000-8000步：2倍增长
- 8000-16000步：1倍增长
- 16000步以上：0.5倍增长

### 用法

1. 在任意聊天窗口输入命令控制功能
2. 每日0点会自动重置步数
3. 功能状态会自动保存，重启插件后保持上次设置

### 命令

#### 功能控制命令
- `/时间步数开` - 开启定时自动增加步数功能
- `/时间步数关` - 关闭定时自动增加步数功能
- `/消息步数开` - 开启消息自动增加步数功能
- `/消息步数关` - 关闭消息自动增加步数功能

#### 手动控制命令
- `/改步数 数字` - 手动修改步数为指定数值
- `/最大消息步数 数字` - 修改消息步数最大值（手动设置）
- `/分钟步数范围 最小值-最大值` - 修改每分钟步数范围（默认4-9）
- `/保底步数 数字` - 修改保底步数（0表示关闭保底功能，默认为0）

#### 查询命令
- `/步数状态` - 查看当前步数和功能状态
  - 开启消息步数时：显示当前步数、今日目标、进度、保底步数、最大消息步数
  - 开启时间步数时：显示当前步数、保底步数、时间步数范围、每分钟步数范围
  - 都未开启时：显示"步数增加功能未启用，请启用时间步数或消息步数功能。"
- `/步数状态all` - 显示所有功能状态和详细信息

### 注意事项

- 7:00-22:50时间段内自动增加步数，其他时间不增加
- 保底步数功能在22:50时执行，若当前步数小于保底步数则自动修改为保底步数
- 时间步数范围 = 每分钟步数范围 × 步数增加总时间（950分钟）
- 消息步数功能开启时，今日目标和进度跟随最大消息步数
- 如果当天实际步数大于插件增加的步数，会以实际步数为准
- 插件设置会自动保存，重启后保持上次设置
- 时间步数增加依赖微信后台，请保留微信后台，否则微信步数更新不及时
- 消息步数功能基于消息触发，微信在后台时不接收消息，进入前台后一次性处理
