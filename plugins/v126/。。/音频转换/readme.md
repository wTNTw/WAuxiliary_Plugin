
# 音频转换工具使用说明
- **若有疑问请联系我** **联系方式TG:@ljh8686**
## 🎯 功能概述
将MP3和SILK格式音频文件相互转换，支持直接发送或保存到本地。

## 📱 启动方式
1. 任意聊天输入框输入:`转换`
2. 点击即可启动工具

## 📂 操作流程

### 第一步：选择文件夹
```

主界面 → 点击"转换" → 浏览文件夹 → 确认选择

```
- **首次使用**：从默认路径 `/storage/emulated/0` 开始
- **浏览操作**：
  - `⬆ 上一级`：返回上级目录
  - `📁 文件夹名`：进入子文件夹
- **确认选择**：点击"使用当前目录"

### 第二步：选择功能模式
三种处理方式可选：

| 模式 | 功能 | 输出位置 |
|------|------|----------|
| **mp3→silk 并发送** | 转换MP3为SILK格式并发送 | 发送给目标联系人 |
| **silk→mp3 保存** | 转换SILK为MP3格式保存 | `/storage/emulated/0/Download/` |
| **直接发送silk** | 直接发送SILK文件（固定1秒时长）| 发送给目标联系人 |

### 第三步：选择具体文件
- 自动筛选对应格式文件（.mp3 或 .silk）
- 点击文件名开始处理

## 💾 文件路径说明

### 输入路径
- 任意可访问的文件夹（通过浏览选择）

### 输出路径
- **MP3文件**：保存到 `/storage/emulated/0/Download/`
- **临时文件**：存储在应用缓存目录，自动清理

## ⚙️ 技术细节
- **SILK发送时长**：默认1秒（可修改代码第135行调整）
- **文件命名**：自动添加时间戳防止重复
- **缓存清理**：每次处理后自动清理临时文件
- **路径记忆**：记住最后一次访问的文件夹

## 🔧 自定义设置
如需修改默认设置，可调整以下常量：
final String OUT_DIR   = "/storage/emulated/0/Download/";  // 保存路径
final String SP_KEY    = "last_folder";                    // 存储键名
int fixedMs = 1000;                                        // SILK发送时长（毫秒）


⚠️ 注意事项

1. 确保有存储权限
2. 支持的文件格式：.mp3, .silk
3. 大文件转换可能需要较长时间
4. 转换过程不要中断应用

🔄 典型使用场景

1. 发送语音消息：将MP3转为SILK发送
2. 备份语音文件：将收到的SILK转为MP3保存
3. 快速转发：直接发送已有的SILK文件
